# Changelog

## 0.0.2

- Raise the default status bar priority so the worktree picker appears before VS Code's built-in branch indicator.
- Replace the status bar item's branch icon with a tree icon to better match the extension.
- Redesign the extension icon as a tree instead of a cross/atom shape.

## 0.0.1

- Initial release.
- Add a status bar worktree picker.
- Add commands to create, switch, open, delete, and refresh Git worktrees.
- Support creating detached worktrees from the current branch or default branch.
